PHP Игра "Калькулятор"
Консольное PHP-приложение, которое генерирует случайные арифметические выражения и проверяет ответы пользователя. Реализовано с учетом приоритета математических операций.

Особенности
Генерация выражений из 4 случайных чисел.

Использование операторов +, -, *.

Честный расчет: корректная обработка приоритета умножения перед сложением/вычитанием.

Интерактивный консольный интерфейс.

Требования
PHP 8.1 или выше.

Composer.

Установка
Глобальная установка (рекомендуется)
Вы можете установить игру глобально, чтобы запускать её из любой папки вашего компьютера:

Bash
composer global require radiculitca/calculator
Локальная установка для разработки
Если вы хотите изучить исходный код:

Склонируйте репозиторий:

Bash
git clone https://github.com/radiculitca/php-calculator-game.git
Перейдите в папку и установите зависимости:

Bash
cd php-calculator-game
composer install
Запуск
После глобальной установки
Просто введите в терминале:

Bash
calculator
При локальной разработке
Из корня проекта выполните:

Bash
php bin/calculator
Использование
После запуска программа выведет математическое выражение.

Введите ваш ответ в консоль и нажмите Enter.

Программа сообщит, был ли ответ верным, и покажет правильный результат в случае ошибки.
