# Лабораторная работа №2: Веб-приложение "Калькулятор"

Веб-приложение на PHP, реализующее игру "Калькулятор" с сохранением результатов в базу данных SQLite.

Приложение построено с использованием паттерна **Page Controller** и стандартных возможностей PHP.

---

## 📝 Описание

Игра проверяет навыки устного счёта.

При каждом запуске генерируется случайное математическое выражение, которое нужно вычислить.

Результаты игры сохраняются в базе данных и доступны для просмотра.

---

## 🎮 Правила игры

1. Генерируется выражение из **4 чисел** и **3 операторов** (`+`, `-`, `*`)
2. Необходимо вычислить результат с учётом приоритета операций
   (сначала умножение, затем сложение и вычитание)
3. Ввести ответ в консоль
4. Программа проверяет результат и сообщает, верно ли решение

---

## ⚙️ Используемые технологии

- PHP (встроенный веб-сервер)
- SQLite (через PDO)
- HTML (с альтернативным синтаксисом PHP)

---

## 🚀 Запуск приложения

1. Перейти в каталог Task02:

```bash
cd Task02
```

2. Запустить встроенный веб-сервер PHP:

```bash
php -S localhost:3000 -t public
```

3. Открыть в браузере:

```
http://localhost:3000/
```

---

## 🗄️ База данных

Приложение использует SQLite-базу:

```
db/calculator.sqlite
```

Таблица `games` содержит:

- id — id игрока
- player_name — имя игрока
- expression — выражение
- correct_answer — правильный ответ
- user_answer — ответ пользователя
- is_winner — результат (победа/поражение)
- game_date — дата и время игры

---
