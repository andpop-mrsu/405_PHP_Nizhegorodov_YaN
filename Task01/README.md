# Лабораторная работа №1: Консольная игра "Калькулятор"

Интерактивное консольное приложение на PHP, опубликованное как независимый пакет на Packagist.

---

## 🔗 Ссылки на проект

* **Packagist:** https://packagist.org/packages/radiculitca/calculator
* **GitHub Repository:** https://github.com/radiculitca/php-calculator-game

---

## 📝 Описание

Игра проверяет навыки устного счёта.

При каждом запуске генерируется случайное математическое выражение, которое нужно вычислить.

---

## 🎮 Правила игры

1. Генерируется выражение из **4 чисел** и **3 операторов** (`+`, `-`, `*`)
2. Необходимо вычислить результат с учётом приоритета операций
   (сначала умножение, затем сложение и вычитание)
3. Ввести ответ в консоль
4. Программа проверяет результат и сообщает, верно ли решение

---

## 🚀 Установка

### Глобальная установка (рекомендуется)

```bash
composer global require radiculitca/calculator
```

После установки игру можно запускать из любой директории.

---

### Локальная установка

```bash
git clone https://github.com/radiculitca/php-calculator-game.git
cd php-calculator-game
composer install
```

---

## ▶️ Запуск

### Глобально

```bash
calculator
```

---

### Локально

```bash
php bin/calculator
```
